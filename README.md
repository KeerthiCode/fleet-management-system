# Fleet Management System

A full-stack Fleet Management System project developed using Spring Boot and MySQL.

## Technologies Used

- Java
- Spring Boot
- MySQL
- HTML
- CSS
- JavaScript
- GitHub

## Features

- Add Drivers
- Manage Vehicles
- REST API Integration
- MySQL Database Connection
- Frontend Dashboard

## Project Structure

backend/
frontend/

## Run Project

1. Start MySQL
2. Run Spring Boot application
3. Open frontend/index.html
